The Portfolio of Vadim Monakhov

Projects

  • The Legend of Zelda (Unity Remaster, First Dungeon)
    Led the development of key gameplay systems in Unity using C#, including reverse engineering NPC AI mechanics and implementing authentic grid based movement.
    Additionally, introduced a new original game changing feature: "Turn Based Combat", inspired by classic RPGs like Dungeons & Dragons, that pivots reactive fast paced authentic Zelda gameplay into a more strategic and tactical experience.
  • Nightfall
    Developed a shoot 'em up roguelike video game in Unity using C#. The game was inspired by Vampire Survivors and features a unique blend of fast-paced action and strategic gameplay. In contrast with classic 2D shoot em' ups this game is 3D and features a jump mechanic alongside with enemies that can merge into one big boss if player does not eliminate them quick enough!

  • VEngine2D
    Developed a Cross-Platform 2D video game engine powered mainly by C++, SDL2 and Box2D (for realistic physics). VEngine game developers use Lua scripting layer to define behavior of their games. Other third party libraries utilized by the engine include: GLM, LuaBridge, and rapidjson. The engine supports Windows, Linux, Mac and Web builds.
